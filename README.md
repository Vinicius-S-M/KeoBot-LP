# 🤖 KeoBot Landing Page

Landing page do **KeoBot**, um robô de desktop focado em organização de tarefas e produtividade.

Este projeto foi desenvolvido como minha primeira landing page, com foco em prática de front-end, estruturação de layout e design de interface.

---

## 🚀 Sobre

O KeoBot é apresentado como um assistente inteligente que ajuda a gerenciar tarefas de forma eficiente, com uma interface moderna e simples.

A página inclui seções como:

- Introdução ao produto  
- Features  
- Comunidade  
- Navegação básica  

---

## 🛠️ Tecnologias

- HTML  
- CSS
